# Operations Analytics Project

Work in progress.

This project aims to analyze an e-commerce dataset using SQL to calculate typical e-commerce KPIs and extract operational business insights.

The dataset contains four tables:
- customers
- products
- transactions
- clickstream

The original dataset includes large tables (transactions ~250MB and clickstream ~1.6GB).  
To keep the repository lightweight, sampled versions of the datasets are included while preserving the original schema.

At this stage the repository only contains the dataset samples.  
The next steps of the project will focus on calculating e-commerce KPIs and analyzing the purchase funnel.
